# Semantic Commit Messages

Format: `<type>(<scope>): <subject>`

```
00.00.00
^   ^  ^
|   |  |
|   |  +-> "FEAT: <commit message>"  <--- Update the patch version
|   |
|   +----> "FIX: <commit message>"   <--- Update the minor version  
|
+--------> "CHORE: <commit message>" <--- Update the major version
```
